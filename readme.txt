Adilo Embed wordpress plugin to support oembed url in wordpress

== Description ==

Adilo's embed plugin adds oEmbed support so
that pasting a link of the video's page in Adilo will embed the video.

== Automatic Installation ==

1. In Admin Dashboard, go to Plugins > Add New.
2. Search for "Adilo Embed".
3. Find the plugin and click "Install".

== Manual Installation ==

1. Make a 'adilo-embed' directory in '/wp-content/plugins/'.
2. Upload all files in the repository to the '/wp-content/plugins/adilo-embed/' directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
